Abstract

Este trabalho consiste na construção de um servido rque permite utilizar as operações CRUD num arquivo sonoro

Para esse efeito foi criado um ficheiro JSON cancoes.json para ser o armazenamento do arquivo.

Para adicionar um id a cada música foi utilizado nanoid na função definida no ficheiro JS adicionar.js

As páginas HTML são geradas a partir de ficheiros pug.

  • Página usada para visualizar uma arquivo sonoro (PUG)
