Abstract

Este trabalho consiste na criação de um executável com a mesma funcionalidade do comando mongoImport, ou seja, capaz de importar ficheiros JSON para o Mongo.

Para esse efeito foi criado um esquema vazio com uma estrutura flexivel e a partir dele é criado um modelo com o nome da coleção.

O modelo é utilizado para guardar os dados obtidos a partir da leitura de um ficheiro JSON

Para execuctar o executavel:

node mongoImport.js Nome_Da_BD Nome_da_Coleção Nome_do_Ficheiro
