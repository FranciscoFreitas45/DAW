Abstract

Este trabalho consiste na construção de um servidor para o website dos arqueossítios.

Para esse efeito foi criada uma stylesheet que processa o ficheiro arq.xml e gera um ficheiro por cada arqueossitio.Estes ficheiros são guardados numa pasta.

Para além disso, foi gerado um índice organizado por concelhos.

A cada ficheiro XML foi associado uma stylesheet XSLT que o transforma para HTML

Por último, foi construido o servidor em nodejs que responde a pedidos na porta 7777 da seguinte forma:

localhost:7777/3 --> Coloca no browser o conteúdo do arq3.xml em formato HTML
