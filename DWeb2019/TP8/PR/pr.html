Abstract

Este trabalho consiste na realização de dois exercícios que abordam o tema indicado

No primeiro exercicio era pedido para fazer a a conversão do dataset disponibilizado em página para um formato arbitrário. Foi utilizado o formato JSON para armazenar essa mesma informação. Este ficheiro foi posteriormente importado para o MongoDB (através do comando mongoimport)

O segundo exercício consistia na criação de um servidor que permitia gerir a base de dados em causa e, ainda, suportar uma determinada API de dados. Nesta última contemplam-se vários pedidos GET sobre a informação presente na base de dados, nomeadamente a lista de todos os prémios Nobel, os dados relativos a um determinado prémio Nobel ou ainda a lista das categorias dos mesmos. Como extra foi também implementada interface Web sobre este servidor
